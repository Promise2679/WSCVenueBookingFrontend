# WSCVenueBookingFrontend

场地管理系统的前端部分

后端仓库：https://github.com/MucheXD/WSCVenueBookingBackend

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
